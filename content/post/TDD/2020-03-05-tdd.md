## 실패

2020-03-05

```javascript
const solution =  (n) => {  
  const changeBinary = n.toString(3);
  const arr = changeBinary.split('');

  const changeNumberByRule = arr.map(v =>
    v == 0 ? v = 1
    : v == 1 ? v = 2
      : v == 2 ? v = 4 : v
    );
  
  return parseInt(changeNumberByRule.join(''));
};
```
