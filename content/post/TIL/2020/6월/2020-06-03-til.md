## Findings

- charAt(i) 메서드에 대해 알게되었습니다. charAt(i) 메서더는 문자열에서 캐릭터를 뽑아줍니다.  
  
  ```java
  String str = "Hello";
  charAt(0); // 'H'
  charAt(1); // 'e'
  charAt(2); // 'l'
  charAt(3); // 'l'
  charAt(4); // 'o'
  ```
